# Malkier

Agent harness and UI using Effect & SolidJS

## Memory

Log your activities with the `memory-manager` skill to keep track of what you're doing. Without this you will not remember anything during the next session with the User. Your memory will help you learn the user's tastes, learn from past mistakes, and save time when working on this project.

## Tasks

Use the `continuum-task` skill to keep track of medium to large tasks that may lead to long sessions or context windows. It has a way to help you plan and track implementation steps. Use it to help you stay organized and maintain focus on the final outcome.
